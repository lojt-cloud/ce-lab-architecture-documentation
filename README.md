# Week 2 Architecture Documentation

Documentation of the multi-tier AWS architecture built across Week 2.
A bastion-gated setup with public web tier, private app tier, and private
database tier, secured with security-group-referenced rules rather than
open IP ranges.

## Contents
- `ARCHITECTURE.md` - detailed component and network description
- `architecture-diagram.png` - visual diagram
- `SECURITY.md` - full security group rules and reasoning
- `COSTS.md` - cost breakdown and optimization analysis
- `IMPROVEMENTS.md` - proposed next steps
